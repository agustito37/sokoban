Sokoban

Welcome

Sokoban is a popular puzzle game developed by Hiroyuki Imabayashi in 1981. The goal is to move all boxes on screen to their designated checkpoints.

Instructions

Use the arrow keys on the keyboard to move the player around the grid. The next level will load when all boxes are placed on checkpoints. If you get stuck, click the Reset Level button on the right to restart the level.

About

Sokoban is built using Easel.js, a library for interacting with HTML5's canvas element. Explore the project's code at Github
